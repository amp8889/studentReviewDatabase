Class Description


{% if course == 'BIOL 330 ~ Professor Martin Wallen'%}

This is an intermediate course in human biology. Practical hands-on labs will engage students' critical thinking and problem-solving skills. Prerequisites include BIOL 104 and BIOL 240.



{% elif course == 'CS 100 ~ Professor Carl Brent'%}

This is an introductory course for computer science majors. Students will learn fundamentals of computers and ethical issues raised by systems.



{% elif course == 'CS 500 ~ Professor Harris Smith'%}

Students will gain an understanding of network performance with respect to several architectures. This course contains multiple assignments which require use of software tools for network analysis as well. Prequisites include CS 200 and CS 402.



{% elif course == 'PHYS 400 ~ Professor Sophia Willard'%}

Students will gain an understanding of advanced methodologies in physics pertaining to aerodynamics. This course will entail practical labs and collaborative projects. Prequisites include PHYS 200 and PHYS 332.



{% elif course == 'MATH 500 ~ Professor Laura Hill'%}

Students will gain an understanding of advanced methodologies in mathematics pertaining to differential equations and linear algebra. This course will entail practical labs and collaborative projects. Prequisites include MATH 300 and MATH 400.



{% endif %}
